# ASPNET-DK24TT80171-LIEUHOANGTRI-DEMO2

## Thông tin đồ án
- Đề tài: Xây dựng Website Bán Sản Phẩm Công Nghệ Apple
- Sinh viên: Liêu Hoàng Trí
- MSSV: DK24TT80171
- Công nghệ: ASP.NET Web Forms, C#, SQL Server

## Cấu trúc thư mục
- setup/: chứa file cài đặt, database, dữ liệu thử nghiệm
- src/: chứa toàn bộ mã nguồn chương trình
- thesis/doc/: chứa file báo cáo Word
- thesis/pdf/: chứa file báo cáo PDF

## Hướng dẫn chạy
1. Mở source trong Visual Studio.
2. Restore database từ file SQL trong thư mục setup.
3. Chỉnh connection string trong Web.config.
4. Chạy project bằng IIS Express hoặc F5.
